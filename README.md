# Simple Drawer
Let's draw something on a blank piece of paper.

## Demo
<a href="https://borecki.github.io/drawer/">live priview available here</a>

#### Used to make this:
* HTML5
* CSS3
* JavaScript
* CANVAS
* Twitter Bootstrap

#### Tools available:
* 28 colors
* brush
* rubber
* clean entire field
* save picture
